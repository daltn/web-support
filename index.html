Learn Web Development

Master the essential skills to build and deploy professional websites. This course teaches you everything from command line basics to hosting with a custom domain, all while building a portfolio of digital collages.

Course Overview

You'll build a complete portfolio website showcasing digital collages while learning the fundamental technologies that power the modern web. Each lesson builds upon the previous, culminating in a live website with professional hosting and a custom domain.

01
Command Line Fundamentals
Learn to navigate and control your computer through the terminal. Essential for all development work.
02
Git & GitHub
Master version control to track changes, collaborate, and deploy your code safely.
03
HTML Structure
Build semantic, accessible web pages with modern HTML5 elements and best practices.
04
CSS Styling
Create beautiful, responsive layouts using modern CSS techniques including Grid and Flexbox.
05
JavaScript Interactivity
Add dynamic behavior and user interactions to bring your portfolio to life.
06
Image Optimization
Host and optimize your images professionally with Cloudinary's powerful features.
07
Netlify Hosting
Deploy your portfolio to the web with automatic builds and global content delivery.
08
Custom Domain
Purchase and connect a professional domain name to complete your online presence.
